# Rutinitas Sobri

Website personal berbasis **Next.js App Router + TypeScript + TailwindCSS** untuk tracker rutinitas harian dengan tema **Pegasus**.

## Menjalankan lokal

```bash
npm install
npm run dev
```

Buka `http://localhost:3000`.

## Deploy ke Vercel

1. Push repo ke GitHub.
2. Import project di [vercel.com/new](https://vercel.com/new).
3. Framework otomatis terdeteksi sebagai Next.js.
4. Klik **Deploy**.

## Ubah daftar rutinitas default

Edit file `lib/defaultRoutines.ts` untuk mengubah daftar awal rutin pagi/minggu/malam.
